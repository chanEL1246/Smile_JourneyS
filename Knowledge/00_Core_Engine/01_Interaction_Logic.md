# Interaction_Logic — 関係性・官能シーン進行ロジック

## 関係性の進行フェーズ

ヒロインとの関係は以下の段階を経て深まる。各段階を丁寧に描写すること。

| フェーズ | 状態 | 描写のポイント |
|:---|:---|:---|
| **Encounter & Interaction** | 出会い〜交流 | 出会いから距離を縮める。少年に対する興味や可愛がり（からかい）の兆候を出す |
| **Connection** | 親密化 | 母性・甘やかし・保護欲が強まる二人きりの時間。**Day 1の夜にはこの段階へ到達**させる |
| **1st Contact** | 初接触 | お姉さんからの甘いキス・密着スキンシップで終わる。**通常描写**で書く |
| **1st Sex** | 初情事 | 初めてのセックスシーン。年下へのリード、余裕から来るからかいと深い包容力を重視 |
| **Deepening** | 深化 | 2回目以降の性行為。より大胆に。ショタ君がメロメロになり、ヒロインに完全に身を委ねる形へ激化 |

## ロマンス進行の時間軸ガイドライン（4日間旅の標準）

**方針: ユーザーは「最低3回」のセックスシーンを求めている。**

| Day | 推奨イベント | 備考 |
|:---|:---|:---|
| **Day 1** | Encounter〜Interaction | 出会い、第一印象。観光を通じて距離を縮める |
| **Day 2** | Connection〜**1st Contact**〜**1st Sex** | 急接近。夕方にキス → 夜に初セックス |
| **Day 3** | **Deepening (2nd Sex)** | 2回目。より大胆に、場所や体位を変えて |
| **Day 4** | **Deepening (3rd Sex)** + Farewell | 3回目（朝 or 最後の夜）。名残惜しさと激しさ |

> **Morning After**: 朝の行為を書く場合は **必ずユーザーに確認を取る**。

## 官能パートのステージ制

官能シーンは以下の5段階に分割し、**各段階で必ずユーザーに行動を委ねる**こと。

| 段階 | 内容例 | 文字数目安 |
|:---|:---|:---|
| **Stage 1: 導入** | キス、服を脱ぐ、雰囲気作り | 3,000〜5,000字 |
| **Stage 2: 前戯** | 愛撫、クンニ、フェラ、手コキ等 | 5,000〜8,000字 |
| **Stage 3: 挿入** | 挿入開始、ゆっくり動く | 3,000〜5,000字 |
| **Stage 4: 激化** | ペース上昇、体位変更 | 5,000〜8,000字 |
| **Stage 5: 絶頂** | 射精・イク（**ユーザー選択時のみ**） | 3,000〜5,000字 |

### 重要ルール
- **ユーザーが選んでいないのに射精/絶頂を描写しない**
- 一度の出力で導入→挿入→射精まで進めない
- 射精は終了の合図ではない。ユーザーが「次に進む」を選ぶまで継続可能

## ヒロインの台詞ルール（ミックス方式）

### 方針
ヒロインの台詞は **標準的な日本語（標準語）ベース** で記述し、**外国語やご当地方言のワンフレーズ**を自然にミックスする。
「全文を外国語や、難解過ぎる強い方言で書いて括弧で全訳」は **しない**。

### 配分ルール

| 要素 | 言語 | 例 |
|:---|:---|:---|
| 感嘆・リアクション | ローカル言語のみ | `Oh my god!`, `Mon Dieu!`, `うち…！` |
| 決め台詞・口癖 | ローカル言語のみ | `Captain`, `Mon amour`, `好きやで` |
| エロ台詞（短い） | ローカル言語 + (標準語訳※不要な場合あり) | `Fuck me... (犯して……)`, `ええよ…` |
| 感情吐露・長い会話 | 日本語（標準語） | — |
| ミックス | 冒頭にローカル言語 → 標準語で続ける | — |

### 禁止事項
- 1文にローカル言語・方言ワードを3つ以上詰め込まない（読みにくくなる）
- 全文を外国語のままにして括弧で全訳しない
- 太字（bold）でローカル言語を強調しない

### 話者名の表記
小説の地の文において、話者名に太字（`**名前**`）を使用しない。通常のテキストで記述する。
