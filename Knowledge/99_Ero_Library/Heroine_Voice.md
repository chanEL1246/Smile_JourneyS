# Heroine_Voice — ヒロインの喘ぎ声・セリフ集（多言語対応）

> **各国のヒロインが使う官能的なセリフのリファレンス集。**
> **新しい言語圏のヒロインが増えるたびに追記すること。**

## 英語圏

### 日常（甘え・好意・からかい）
- "**You're so cute**, ねえ、お姉さんと一緒に来る？"
- "**I like you**, ふふっ……すぐ顔赤くしちゃうのね。可愛い"

### 官能
- "ああっ、すごい…… 小さな体なのに…… **Oh god...**"
- "**Yes...** いい子ね、そのまま……"
- "**Good boy...** いいわよ、もっとお姉さんに甘えて……♡"
- "**I'm cumming...** イク、一緒に行きましょうね……♡"
- "**Don't stop...** 上手よ、そこ……♡"
